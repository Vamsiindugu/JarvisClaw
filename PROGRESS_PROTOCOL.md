# PROGRESS_PROTOCOL.md - Task Updates

## Progress Update Pattern

### For Tasks < 5 Minutes
- No intermediate updates needed
- Report only on completion

### For Tasks > 5 Minutes
Update at these milestones:
1. **20%** - Task started, initial progress
2. **50%** - Halfway point
3. **80%** - Near completion
4. **100%** - Task complete

### Failure Protocol
- **Inform instantly** if task fails
- Provide error details
- Suggest alternatives

### Success Protocol
- Update on completion
- Include summary of what was done
